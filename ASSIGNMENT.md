# Live Coding Exercise: Build RESTful CRUD APIs in Go

## Overview

In this assignment, you will build RESTful APIs using **Go**, **Gin**, **GORM**, and **PostgreSQL** that manages a `User` resource. 

---

## Tech Stack

- **Language:** Go
- **Web Framework:** [Gin](https://github.com/gin-gonic/gin)
- **ORM:** [GORM](https://gorm.io/)
- **Database:** PostgreSQL (Bonus: running via Docker)

---

## Requirement

Create CRUD APIs for user management. 
1. Create a new user
2. List all users
3. Get a user by ID
4. Update a user
5. Delete a user

Each user contains ID, name, email. 
You may clarify on the requirements. If you are not familiar with a certain library, you may check on the documentation. 
Think aloud when you work on the exercise. 

---

## Deliverables
- Demo the APIs to assessors
- Source code pushed to a Git repository and share with assessors

You need to share your screen throughout the live coding exercise.
